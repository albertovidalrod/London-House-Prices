What is Property Log?

Property Log is a Google Chrome Extension that works with the Rightmove website to show the price change history of a property. Please note, we are in no way affiliated with Rightmove and this is a 3rd party extension.

How does it work?

Every time a user views a property on Rightmove they check for a change in price. If the price has been updated we log it and use this data to populate the price change history displayed on the page.
